# NovyWave

**NovyWave** is a modern, open-source waveform viewer designed for digital design verification and analysis. It serves as a contemporary alternative to GTKWave, built with Rust for performance and reliability.

![NovyWave Dark Theme](../novywave_dark_linux.png)

## Key Features

- **Multi-Format Support**: Load VCD, FST, and GHW waveform files
- **Cross-Platform**: Runs on Linux, macOS, and Windows
- **Modern UI**: Clean, responsive interface with dark and light themes
- **High Performance**: Efficient handling of large waveform files
- **Keyboard-Driven**: Comprehensive keyboard shortcuts for power users
- **Auto-Updates**: Built-in update system keeps you current

## Who is NovyWave For?

NovyWave is designed for:

- **Hardware engineers** working on digital design verification
- **Students** learning digital design and HDL simulation
- **Hobbyists** exploring FPGA and ASIC development
- **Researchers** analyzing simulation results

## Supported Simulators

NovyWave works with waveform files from any simulator that outputs standard formats:

| Format | Simulators |
|--------|------------|
| **VCD** | Most Verilog/SystemVerilog simulators (Icarus Verilog, Verilator, ModelSim, etc.) |
| **FST** | Verilator, GTKWave-optimized output |
| **GHW** | GHDL (VHDL simulator) |
