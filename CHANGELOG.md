# Changelog

All notable changes to NovyWave will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- GHW file format support (GHDL waveform files)
- GitHub Actions CI/CD release pipeline
- Auto-updater support (tauri-plugin-updater)

### Changed
- File picker dialog now shows "(*.vcd, *.fst, *.ghw)"

## [0.1.0] - 2024-XX-XX

### Added
- Initial release
- VCD and FST waveform file support
- Multi-file loading with smart labeling
- Hierarchical scope navigation
- Variable selection with search/filter
- Timeline visualization with zoom/pan
- Multiple signal formatters (Hex, Binary, Decimal, ASCII, etc.)
- Dark and Light themes
- Dock modes (Right/Bottom panel layout)
- Keyboard shortcuts for navigation
- Configuration persistence (.novywave files)
- Cross-platform support (Linux, macOS, Windows)
